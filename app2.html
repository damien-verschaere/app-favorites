PUMP IT UP AND CHILL

Incredibox est une application de création musicale dans laquelle l'utilisateur peut créer un mix en gérant un groupe de sept beatboxers animés. Une fois l'une des huit versions (ou styles musicaux) disponibles choisie, le lecteur dispose d'une interface composée de sept caractères vierges et de vingt icônes sonores.

Ces icônes sont réparties dans les catégories suivantes (cinq pour chacune d'entre elles) : rythmes, effets, mélodies et voix. Chaque icône est une boucle sonore a cappella unique à glisser-déposer sur les personnages (qui représente le compositeur musical dans toutes les versions) à partir d'une gamme de chapeaux, lunettes de soleil, masques, écouteurs ou autres articles qui les habillent comme ils sont utilisés et laissez-les chanter en rythme les uns avec les autres.
